# 📍 기능 목록

### UI

- [ ] 로또 구입 금액을 입력 받는다.
- [ ] 당첨 번호와 보너스 번호를 입력 받는다.
- [ ] 발행한 로또 수량 및 번호를 출력한다.
- [ ] 당첨 내역을 출력한다.
- [ ] 수익률을 출력한다.
- [ ] 예외 상황 시 에러 문구를 출력한다.

### 도메인 로직

- [ ] 구입 금액에 해당하는 만큼 로또를 발행한다.
- [ ] 사용자가 구매한 로또 번호와 당첨 번호를 비교한다.
  - [ ] 당첨 여부를 판단한다.
  - [ ] 수익률을 계산한다.
- [ ] 사용자가 잘못된 값을 입력하면 예외를 발생시키고 종료한다.
